**🔍 ZoomIt Quick Reference Guide - Essential Shortcuts for Presentations & Demos**

**🚀 The 4 Main Modes**

- **Ctrl + 1 (Zoom):** Freezes the screen and zooms. Use **Mouse Wheel** to adjust. Left-click to draw.
- **Ctrl + 2 (Draw):** Enter drawing mode without zooming.
- **Ctrl + 3 (Timer):** Full-screen countdown. Use **Arrows** to add/subtract time.
- **Ctrl + 4 (Live Zoom):** Zoom in but keep windows interactive (you can still type/click).

**🎨 Drawing & Shapes**

_First, enter Draw Mode (Ctrl + 2) or Zoom Mode (Ctrl + 1)._

| **Feature**          | **Key Command**    |
| -------------------- | ------------------ |
| **Straight Line**    | Hold Shift         |
| **Rectangle**        | Hold Ctrl          |
| **Ellipse / Circle** | Hold Tab           |
| **Arrow**            | Hold Ctrl + Shift  |
| **Change Pen Width** | Ctrl + Mouse Wheel |

**🌈 Color Swatches**

_Tap the letter key while in Draw Mode:_

- **R** = Red | **G** = Green | **B** = Blue
- **O** = Orange | **Y** = Yellow | **P** = Pink

**🖥️ Screen Controls**

- **T (Type):** Press T to start typing text. (Wheel to change size).
- **W (Whiteboard):** Turn the screen into a blank white canvas.
- **K (Blackboard):** Turn the screen into a blank black canvas.
- **E (Erase):** Clears all drawings on the screen.
- **Ctrl + Z:** Undo the last drawing action.
- **Ctrl + S:** Save the current screen (with drawings) as an image.
- **Esc / Right-Click:** Exit current mode.

**📹 Advanced (New Versions)**

- **Ctrl + 5:** Start/Stop Screen Recording.
- **Ctrl + 6:** Snipping Tool (Select a specific area to capture).
- **Shift + Ctrl + 6:** Snipping Tool with file save option.
